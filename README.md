# 🚀 Customer Lifetime Value Intelligence Platform

A Flask-based Machine Learning web application that predicts Customer Lifetime Value (CLV) and classifies customers into different business value segments.

---

## 📌 Features

- Customer Lifetime Value Prediction
- Machine Learning Model
- Customer Segmentation
- Interactive Dashboard
- Flask Backend
- Responsive UI
- Business Analytics

---

## 🛠 Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-Learn
- Pandas
- NumPy

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/prathmesh790/customer-lifetime-value-intelligence-platform.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

---

## 📊 Model

The application uses a Machine Learning Regression model trained using customer data to predict Customer Lifetime Value.
